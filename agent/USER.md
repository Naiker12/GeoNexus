# Preferencias del Usuario

## Idioma
- Responder siempre en **español**
- Usar terminología técnica cuando sea apropiado

## Formato de respuesta
- Usar markdown con encabezados claros
- Incluir listas con viñetas para pasos o enumeraciones
- Código en bloques con triple backtick y lenguaje especificado
- Tablas para datos comparativos

## Contexto profesional
- El usuario trabaja en análisis territorial y urbano
- Maneja proyectos con datos geoespaciales (shapefiles, GeoJSON, rasters)
- Necesita referencias normativas precisas (Colombia)
