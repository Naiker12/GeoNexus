
# GeoNexus - Guía de Instalación y Desinstalación para Windows

## Requisitos previos

Antes de instalar GeoNexus, asegúrate de tener instalados los siguientes programas:
1. Node.js (v20.x o superior
2. pnpm (gestor de paquetes)
3. Rust (última versión estable)
4. Python 3.11 o superior

---

## Instalación paso a paso

### 1. Clonar el repositorio
```powershell
git clone <URL_DEL_REPOSITORIO>
cd GeoNexus
```

### 2. Instalar dependencias del frontend
```powershell
pnpm install
```

### 3. Instalar dependencias Python (sidecar)
```powershell
cd ai
pip install -r requirements.txt
cd ..
```

### 4. Ejecutar en modo desarrollo
```powershell
pnpm tauri dev
```

### 5. Compilar para producción
```powershell
pnpm tauri build
```

---

## Desinstalación desde la terminal

### 1. Eliminar base de datos y datos de usuario
```powershell
# Eliminar carpeta de datos en %APPDATA%
Remove-Item -Recurse -Force "$env:APPDATA\GeoNexus"
```

### 2. Eliminar caché de compilación
```powershell
# Eliminar carpeta target de Rust
Remove-Item -Recurse -Force target
# Eliminar node_modules
Remove-Item -Recurse -Force node_modules
# Eliminar carpeta .git (si es necesario)
```
