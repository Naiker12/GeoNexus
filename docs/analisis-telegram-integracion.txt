5. FLUJO DE USO
================================================================================
1. Usuario configura bot token en la UI
2. Backend inicia polling de Telegram API
3. Usuario envía mensaje al bot en Telegram
4. Backend valida usuario → envía mensaje a GeoNexus
5. GeoNexus genera respuesta → backend la envía a Telegram
6. Usuario recibe respuesta en el chat
