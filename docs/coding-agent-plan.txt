# GeoNexus Coding Agent Enhancement Plan

## 1. Core Features to Implement
- Toggleable "Coding Agent Mode" (on/off switch in UI)
- Real-time coding side panel/sheet with:
  - File tree explorer
  - Live code preview
  - Thought process/timeline (similar to Trae's thought display)
- Integration with chat:
  - Agent is aware when coding mode is active
  - Asks clarifying questions
  - Updates the side panel in real-time
- Make the app feel more like a professional desktop IDE

## 2. File Structure Changes

### New Files to Create:
- `src/components/chat/CodingAgentPanel.tsx` - The main coding agent side panel
- `src/components/chat/CodingThoughtBubble.tsx` - Shows agent's thought process
- `src/components/chat/CodeFileEditor.tsx` - Inline code editor/viewer
- `src/components/chat/FileTreeExplorer.tsx` - Interactive file tree
- `src/components/chat/RealTimePreview.tsx` - Live preview of created apps
- `src/contexts/CodingAgentContext.tsx` - State management for coding mode
- `src/types/coding-agent.ts` - Type definitions for coding agent

### Files to Modify:
- `src/components/chat/ChatPanel.tsx` - Add toggle for coding mode, integrate CodingAgentPanel
- `src/components/chat/ChatComposer.tsx` - Update ToolMenu integration
- `src/components/chat/ToolMenu.tsx` - Finalize removal of "Carpeta Local", add coding mode toggle
- `src/App.tsx` - Add any app-wide state for coding mode if needed
- `src/features/workspace/GeoAgentsWorkspace.tsx` - Ensure workspace integrates with coding mode

## 3. Detailed Implementation Steps

### Step 1: Create Type Definitions
- Create `src/types/coding-agent.ts`
- Define:
  - CodingAgentState (active/inactive)
  - FileNode (for file tree)
  - CodingStep (for thought process)
  - PreviewState (for live preview)

### Step 2: Create Coding Agent Context
- Create `src/contexts/CodingAgentContext.tsx`
- Expose:
  - `isCodingModeActive`
  - `toggleCodingMode`
  - `currentFiles`
  - `addFile`
  - `updateFile`
  - `codingSteps`
  - `addCodingStep`
  - `previewUrl`
  - `setPreviewUrl`

### Step 3: Create File Tree Explorer
- Create `src/components/chat/FileTreeExplorer.tsx`
- Features:
  - Expand/collapse folders
  - Show file icons
  - Click to open file in editor
  - Real-time updates as files are created

### Step 4: Create Code Editor/Viewer
- Create `src/components/chat/CodeFileEditor.tsx`
- Features:
  - Syntax highlighting (use react-syntax-highlighter or similar)
  - Read-only initially (future: editable)
  - Shows file name and content
  - Updates in real-time

### Step 5: Create Thought Bubble Component
- Create `src/components/chat/CodingThoughtBubble.tsx`
- Features:
  - Shows agent's thinking process
  - Animated appearance
  - Shows statuses like "Analizando objetivo...", "Creando archivo...", "Generando preview..."
  - Similar to Trae's thought display

### Step 6: Create Real-Time Preview
- Create `src/components/chat/RealTimePreview.tsx`
- Features:
  - Iframe for previewing created apps
  - Loading states
  - Refresh button
  - Responsive design

### Step 7: Create Main Coding Agent Panel
- Create `src/components/chat/CodingAgentPanel.tsx`
- Combines:
  - Header with toggle and status
  - Thought bubble area
  - File tree explorer
  - Code editor
  - Preview iframe
- Uses Tailwind for styling
- Collapsible/expandable

### Step 8: Integrate with Chat Panel
- Modify `src/components/chat/ChatPanel.tsx`
- Import and use CodingAgentContext
- Add toggle button for coding mode
- Render CodingAgentPanel when mode is active
- Make chat aware of coding mode

### Step 9: Update Tool Menu
- Modify `src/components/chat/ToolMenu.tsx`
- Keep "Workspace IDE" (but maybe rename or adjust)
- Remove any remaining "Carpeta Local" references
- Add toggle for coding mode

### Step 10: Polish and Test
- Style everything to match GeoNexus design
- Test all features
- Make sure the app feels like a real desktop IDE
