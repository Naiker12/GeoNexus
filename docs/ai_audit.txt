3. PROTOCOLO WEBSOCKET
================================================================================

  Cliente → Servidor:
  {
    "action": "<action_name>",
    "params": { ... },
    "session_id": "<opcional>"
  }

  Servidor → Cliente (no streaming):
  {"type": "result", "data": <any>}
  {"type": "error", "message": "<text>"}

  Servidor → Cliente (chat streaming):
  {"type": "delta", "content": "token"}
  {"type": "thinking", "content": "reasoning"}
  {"type": "done", "status": "ok", "message": {...}, "usage": {...}}
  {"type": "error", "message": "<text>"}
