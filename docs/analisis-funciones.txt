3. FUNCIONES MÁS IMPORTANTES DEL SISTEMA
================================================================================

★ PRIORIDAD CRÍTICA (núcleo del sistema):

  1. send_message        → Pipeline completo de chat (clasificación, RAG, tools, grafo)
  2. index_document      → Indexación documental (extraer → chunk → embed → grafo)
  3. recall_chunks       → Búsqueda RAG vectorial
  4. sync_local_connector → Sincronización de conectores locales
  5. list_llm_models     → Auto-detección de modelos LLM
  6. send_message (frontend) → useChatSession hook

★ PRIORIDAD ALTA (operaciones core):

  7. list_data_assets / get_data_asset → CRUD de activos
  8. register_local_connector → Registro de conectores
  9. upload_asset_file → Subida de archivos
  10. dispatch_container_tool → MCP containers
  11. list_graph_nodes / list_graph_edges → Visualización del grafo
  12. ping_llm_provider → Health check de LLMs
  13. get_mentionable_sources → Sistema de @menciones

★ PRIORIDAD MEDIA (análisis y métricas):

  14. get_analysis_metrics → Dashboard de análisis
  15. get_token_timeline → Timeline de tokens
  16. get_model_usage → Uso por modelo
  17. get_top_queries → Consultas frecuentes

★ PRIORIDAD BAJA (utilidades):

  18. open_folder_picker / open_file_picker → Diálogos nativos
  19. read_file_base64 → Lectura de archivos
  20. list_directory → Exploración de directorios
