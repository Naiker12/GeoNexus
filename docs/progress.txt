## Key Decisions
- **Stateless gateway per call**: Simpler than multiplexing — no mutex, no persistent connection, no reconnect logic. Overhead of TCP handshake per call is acceptable for LLM chat frequency.
- **`probe()` before spawn**: Prevents duplicate gateway instances and "port in use" errors. Matches Hermes approach of detecting existing runtime before provisioning.
