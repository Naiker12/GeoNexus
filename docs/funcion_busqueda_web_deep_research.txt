4. PYTHON - WEB SEARCH (ai/web_search.py)
==============================================================================
  search_web(query, provider="duckduckgo", max_results=8):
    Proveedores: duckduckgo | google | bing | serpapi
    DuckDuckGo: usa lite.duckduckgo.com (POST), parsea HTML con regex
    Google: usa Custom Search JSON API (requiere GOOGLE_API_KEY + CX_ID)
    Bing: usa Bing Web Search API (requiere BING_API_KEY)
    SerpAPI: usa serpapi.com (requiere SERPAPI_API_KEY)
    Todos devuelven: { status, sources: [{url, title, snippet, "done"}] }
